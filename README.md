# Flow iOS

Flow is an iOS Nostr client focused on fast feeds, profiles, threads, onboarding, and custom feed creation.

## Requirements

- Xcode 16+
- iOS 18+

## Getting Started

1. Open `Flow.xcodeproj` in Xcode.
2. Build and run the `Flow` scheme.
